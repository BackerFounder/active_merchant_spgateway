## Installation

Add this line to your application's Gemfile:

```ruby
gem 'active_merchant_spgateway',  git: 'git://github.com/BackerFounder/active_merchant_spgateway'
```

And then execute:

    $ bundle

## Usage

僅適用於智付通 1.2 版 API
